# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report privately via [GitHub Security Advisories](https://github.com/Livshitz/appwrap/security/advisories/new). We'll acknowledge your report and work with you on a fix and coordinated disclosure.

## Supported versions

appwrap is pre-1.0; only the latest published version of each package receives security fixes.
